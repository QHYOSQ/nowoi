# nowoi — Free Online Tools Platform

## Overview
nowoi is a clean, modern platform designed to provide free online tools for developers, writers, and content creators. The platform is hosted on GitHub Pages, ensuring fast and reliable access for users around the world.

## Features
- **Homepage**: A navigation hub with featured tools and category links.
- **Online Tools**: A comprehensive collection of tools that can be accessed directly in the browser without any backend dependencies.
- **Responsive Design**: Optimized for both desktop and mobile devices, ensuring a seamless user experience.
- **Privacy Focused**: All tools run locally in the browser, no data is stored on servers.

## Getting Started
1. Clone the repository to your local machine.
2. Open `index.html` in your browser to view the homepage.
3. Navigate through the tools to explore the available features.

## Available Tools
- **Developer Tools**: JSON Formatter, JSON Validator, UUID Generator, Timestamp Converter, HTML Minifier, CSS Minifier, JS Minifier
- **Text Tools**: Word Counter, Case Converter, Slug Generator, Markdown Preview, Remove Line Breaks, Text Diff Checker
- **Encoding Tools**: URL Encoder, Base64 Encoder, Image to Base64
- **Generators**: Password Generator, QR Code Generator, Random Number Generator
- **Other Tools**: Color Picker
