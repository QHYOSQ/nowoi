Quick Reference

Control Characters (0-31): Non-printable characters used for device control (NUL, TAB, LF, CR, etc.)

Printable Characters (32-126): Standard ASCII including letters, digits, punctuation, and symbols

Extended ASCII (127-255): Additional characters including DEL and extended Latin characters

About ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard that represents text in computers and communication equipment. It uses 7 bits to represent 128 characters, including control characters and printable characters.

Common ASCII Codes

  • NUL (0): Null character
  • TAB (9): Horizontal tab
  • LF (10): Line feed
  • CR (13): Carriage return
  • Space (32): Space character
  • 0-9 (48-57): Digit characters
  • A-Z (65-90): Uppercase letters
  • a-z (97-122): Lowercase letters
  • DEL (127): Delete character
