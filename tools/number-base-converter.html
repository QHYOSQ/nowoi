About This Tool

Convert numbers between any bases (2-36). Supports binary, octal, decimal, and hexadecimal conversions. Includes a base calculator for arithmetic (+, ×, ÷) and bitwise operations (AND, OR, NOT). Free, no signup.

This tool runs in your browser; no data is stored.

How to Use

Converter: Select source and target bases. Enter number. See converted result. Copy if needed.

Calculator: Enter two numbers with their bases. Select operation. Choose result base. See calculated result.
